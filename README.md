#CCTL_Project

###Installation

Install [NodeJS](http://nodejs.org/)

Install [MongoDB](http://www.mongodb.org/)

Clone the project:
```
git clone <project>
```
Install dependencies:
```
cd CCTL_Project
npm install
```
 
###Run it

```
cd  node
node server.js
```
The programm will say :
```
server listening on port : <port>
```
Open your browser and go to "localhost:<port>"
